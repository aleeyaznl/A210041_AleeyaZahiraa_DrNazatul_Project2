🍽️ Reflection & Learning Journey

Throughout this course, I progressed from creating simple user interfaces using Jetpack Compose to developing complete Android applications integrated with Room Database, Firebase Firestore, REST APIs, and mobile sensors.


Each laboratory exercise introduced new concepts that gradually expanded my understanding of mobile application development. Beginning with UI design, user interaction, and navigation, I eventually learned how to manage local persistence, cloud synchronization, external data sources, and hardware integration.


Through NutriLink, I gained practical experience in designing solutions that support SDG Goal 2: Zero Hunger. This project allowed me to combine technical skills with social impact by creating a platform that promotes food sustainability and community food sharing.


Overall, this course strengthened my problem-solving, debugging, UI/UX design, and software architecture skills while preparing me for future mobile application development projects.
